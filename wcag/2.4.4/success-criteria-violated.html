Success Criteria Violated?

WCAG Success Criteria

  • 2.4.4 Link Purpose (In Context) Level A:
    The purpose of each link can be determined from the link text alone or from the link text together with its programmatically determined link context, except where the purpose of the link would be ambiguous to users in general.
  • 2.4.9 Link Purpose (Link Only) Level AAA:
    A mechanism is available to allow the purpose of each link to be identified from link text alone, except where the purpose of the link would be ambiguous to users in general.

Fails WCAG 2.4.4 (A)

Below is an example of a real-world failure, where identical non-descriptive link text appears multiple times and the surrounding context is not programmatically associated with the links (e.g., the first column is not marked up as a row header).

Resource Description Responsible Party Full Details
Budget Planning Guide Outlines forecasting methods and cost allocation strategy Finance Strategy Office Read more
Tax Compliance Overview Summarizes obligations for state and federal reporting Tax and Treasury Read more
Financial Aid Services Explains types of aid and eligibility criteria Student Financial Services Read more

Note: These links do not meet WCAG 2.4.4 because a screen reader user navigating by links will hear multiple "Read more" links without any way to quickly distinguish between them. The context in the first column is not programmatically associated with the link text.

Passes WCAG 2.4.4 and WCAG 2.4.9

In this version, the text of the resource name is a link, and it's located in the first column of each row. This approach improves clarity and avoids the need for separate generic link text. Because the purpose of each link can be understand by itself, this example meets both success criterion 2.4.4 and 2.4.9.

Resource Description Responsible Party
Budget Planning Guide Outlines forecasting methods and cost allocation strategy Finance Strategy Office
Tax Compliance Overview Summarizes obligations for state and federal reporting Tax and Treasury
Financial Aid Services Explains types of aid and eligibility criteria Student Financial Services

Note: This version avoids repeated vague link text by using the resource name itself as the link, which is both descriptive and naturally integrated into the table structure.

Passes WCAG 2.4.4 but not WCAG 2.4.9

This version meets WCAG 2.4.4 by associating the topic with each link through proper use of row headers. Although the links still say "Read more," their purpose is clear when assistive technologies associate them with the first cell of each row.

Resource Description Responsible Party Full Details
Budget Planning Guide Outlines forecasting methods and cost allocation strategy Finance Strategy Office Read more
Tax Compliance Overview Summarizes obligations for state and federal reporting Tax and Treasury Read more
Financial Aid Services Explains types of aid and eligibility criteria Student Financial Services Read more

Note: This table passes WCAG 2.4.4 because the first column is marked with scope="row", making it programmatically associated with the "Read more" links in the same row.
